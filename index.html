GW2 Owned Material Crafting Optimizer

Find crafts that turn materials you already own into more gold than selling the materials directly.
Usage Guide

First use

  1. Add one or more GW2 API keys with account, characters, and inventories permissions.
  2. Click Save API Keys. Keys are stored locally in this browser.
  3. Use Run Full Scan for Profit Mode, or Run Research Note Scan for Research Note Mode.
  4. Let the tool load owned materials, recipe cache, TP prices, and then build the table.

Repeat use

  1. Do not reload recipes every time unless you want a fresh recipe cache.
  2. Usually run Load Owned Materials if your inventory changed.
  3. Run Refresh TP Prices when you want newer Trading Post values.
  4. Then click Build Opportunities or Build Research Note Table.
  5. Use Clear Data Cache only when cached recipe/item/price data seems wrong or stale.

Pricing modes

  • Instant Sell values tradeable materials using current buy orders.
  • Sell Listing values tradeable materials using current sell listings.
  • Some vendor materials are forced to Vendor cost even when TP modes are selected.
  • Missing buy orders may fall back to TP Sell and should be labeled that way in material details.

Profit Mode columns

  • Item: crafted output item.
  • Recipe: recipe/source used for the craft.
  • Can Make: how many you can craft from currently detected owned materials.
  • Prof Qty: estimated quantity that can sell into profitable buy-order depth.
  • Craft Qty: recommended craft amount after limits are applied.
  • Avg Sell: average estimated sell value across the profitable depth.
  • Last BO Hit: lowest buy order touched by the estimated profitable quantity.
  • Top BO: current highest buy order for the output.
  • Top BO Net: highest buy order after Trading Post fees.
  • Mat Sell Value: value of the materials consumed by the recipe.
  • Profit Each: extra profit per craft after material value.
  • Profit Total: estimated total extra profit for the recommended craft quantity.
  • ROI: profit compared against consumed material value.
  • Owned Mats Used: ingredient breakdown and pricing source details.

Research Note Mode columns

  • Item: crafted item being checked for research salvage.
  • Rarity: item rarity shown for salvage filtering/reference.
  • Professions: crafting disciplines capable of crafting the item.
  • Recipe: recipe/source used to craft the item.
  • Can Make: number craftable from currently detected owned materials.
  • Craft Qty: craft quantity used for the displayed estimate.
  • Expected Notes / Item: estimated research notes gained per salvaged item. Entries listed as unknown do not have any significant drop data yet.
  • Total Notes: estimated total notes from the displayed craft quantity.
  • Mat Value / Note: material value divided by estimated total notes.
  • Out Qty: quantity produced per craft.
  • Owned Mat Value / Craft Cost: owned-material opportunity value when Use Owned Materials is checked; full acquisition/craft cost when unchecked.
  • TP Buy / TP Sell: output item Trading Post prices, shown when Use Owned Materials is unchecked.
  • Owned Mats Used / Recipe Inputs: ingredient breakdown and pricing source details.
